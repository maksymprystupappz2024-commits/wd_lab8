Тестова адаптивна сторінка про спорт

Тут є типові елементи Bootstrap: навігація, dropdown, двоколонкова структура, кнопки, форма, карточки, список, а також слайдер (carousel). Сторінка адаптивна і добре виглядає на телефоні.

Подивитись галерею Розклад
Mobile friendly
Швидко
Bootstrap 5

Що використано

  • Navbar + Dropdown OK
  • 2 колонки (grid) OK
  • Carousel (слайдер) OK
  • Buttons + Icons OK
  • Form + Validation OK
Адаптивність: спробуй звузити вікно браузера або відкрий DevTools → Toggle device toolbar.

Коротко про види спорту

Футбол

Командна гра, де важливі тактика, витривалість і взаємодія.

Волейбол

Швидкі розіграші, точні передачі та командна підтримка.

Баскетбол

Динамічна гра з кидками, дриблінгом та швидкими атаками.

Підказки до звіту
  1. Скрін: підключення Bootstrap (локальні файли в папці libs/bootstrap).
  2. Скрін: Navbar + Dropdown.
  3. Скрін: Сітка (ліва і центральна колонки) + адаптивність (mobile).
  4. Скрін: Carousel (слайдер) працює.
  5. Скрін: Форма + перевірка (натисни Submit без полів).
  6. Скрін: Публікація на GitHub Pages.
